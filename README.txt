Install Instructions (for local use):
1) Install node.js 
	https://nodejs.org/en/download/
2) Open up command line and navigate to the project directory
3) npm install npm@latest -g
4) npm install
5) npm start

type localhost:8888 into your browser

NOTES

Sometimes the server takes too long to respond when using the app and
the loading icon keeps spinning. If this happens, you have to restart 
the app. :(

Make sure you are on UW campus or using HUSKYONNET

Restart Instructions:
1) Control-c
2) npm start
